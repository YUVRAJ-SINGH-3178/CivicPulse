# 🏛️ CivicPulse – Empowering Citizens, Enabling Governance

Welcome to **CivicPulse**! This is a modern, full-stack web application built to transform how local civic issues—such as potholes, broken streetlights, or waste management—are reported, tracked, and resolved.

By bridging the gap between citizens and municipal authorities, CivicPulse introduces unprecedented transparency, accountability, and community engagement to the local issue resolution process.

## 🚀 Key Features

*   **📍 Seamless Issue Reporting:** Submit detailed reports on civic problems, complete with live location tagging via interactive maps and image uploads for clear context.
*   **🔁 Transparent Status Tracking:** Watch your reports move in real-time from *Open* to *In Progress* to *Resolved*.
*   **👍 Community Prioritization (Upvoting):** Highlight urgent community needs by upvoting reports from other citizens.
*   **🧑‍💼 Admin Dashboard:** A dedicated portal empowering city authorities to filter, manage, and update the status of issues efficiently.
*   **📘 Civic Education Module:** An interactive learning center for students and first-time voters, complete with quizzes and gamified XP to educate users on their civic rights.
*   **🧭 Civic Simulator:** Step into the shoes of a city leader! Tackle animated dilemmas, manage budgets, and make decisions that dynamically affect public satisfaction and resources.

## 🛠️ Technology Stack

*   **Frontend:** React.js, Tailwind CSS, Leaflet.js (for maps), Framer Motion, and Three.js.
*   **Backend:** Node.js, Express.js.
*   **Database:** PostgreSQL / MongoDB (Mongoose).
*   **Security & Auth:** JWT Authentication (Clerk), Helmet.js, Express Rate Limit.
*   **Integrations:** Cloudinary for seamless image hosting, EmailJS for notifications.

## 📥 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YUVRAJ-SINGH-3178/CivicPulse.git
   cd CivicPulse
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Set up Environment Variables:**
   Copy `.env.example` to `.env` and configure your database strings and Cloudinary keys.
4. **Run the Application:**
   ```bash
   npm start
   ```
